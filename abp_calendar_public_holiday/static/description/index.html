Public Holiday on Calendar

Description of the module

Features

Public holidays are showed on the calendar for every internal user

Public holiday on calendar

When you create a public holiday, a calendar event is automatically created.
Calendar automatic update

When the public holiday is modified, the changes are automatically applied on the calendar event.
Add new employee (internal user)

When a new employee join the company, he will be added on all the public holiday calendar events.
Set public holiday calendar event to all day

You can specify if the public holiday is all day, the calendar event will be updated automatically.
