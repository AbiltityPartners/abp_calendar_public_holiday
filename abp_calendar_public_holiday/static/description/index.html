Public Holiday on Calendar Overview

This module allows you to view the public holidays declared in the "Time Off" Apps within the Calendar Apps.

How does it work ?

Discover how it works in our demo video

Feature documentation

1. Create a public holiday

Create a public holiday with a name, start date and end date.
2. View public holiday

The public holiday is displayed in the calendar for all the internal users.
3. Update public holiday information

Changing the name or dates in the public holiday form view affects the event on the calendar.
4. Automatically add new internal user

When adding an internal user, the event gets updated.
